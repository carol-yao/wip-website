No Connection

Development • Summer 2016

STACK

HTML • CSS • JavaScript

OVERVIEW

No Connection is a chrome extension I developed over Summer 2016 when I first started programming. I thought this simple idea would be a great way to learn the basics of chrome extensions before diving deeper in.

I originally got this idea when I was having internet issues at home and kept getting the “No Connection” page displayed and thought it’d be a pretty funny prank to always show this page. The extension icon is a blank image so it is concealed well in the chrome bar. When this extension is turned on, it replaces screens with the No Connection page even if they have internet!
